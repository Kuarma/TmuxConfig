# SETUP

> Install tmux on arch linux
```bash
sudo pacman -S tmux
```
> Install TPM (**T**mux **P**lugin **M**anager)

```bash
git clone https://github.com/tmux-plugins/tpm ~/.tmux/plugins/tpm
```
> Setup tmux with my config 

```bash
git clone https://github.com/Kuarma/TmuxConfig.git ~/.config/tmux/
```
